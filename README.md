# Shukatsu Photo

スマホやPCのブラウザだけで、就活用の証明写真を作成できるWebアプリです。

## 主な機能

- スマホのフロント／リアカメラで撮影
- 端末内の写真を読み込み
- 3:4の就活写真ガイドに沿った位置・拡大調整
- 四隅の色を利用した背景の自動均一化
- 白・ライトブルー・ライトグレーの背景
- 明るさ、コントラスト、自然なスムージング、色温度調整
- Web ES用 1200 × 1600 px、履歴書用 900 × 1200 px
- JPEG／PNGの高画質保存
- PWA対応、2回目以降はオフラインでも利用可能

写真は外部サーバーへ送信せず、ブラウザ上だけで処理します。

## 起動方法

カメラ機能を使うため、`index.html` を直接開かずローカルサーバーで起動してください。

```powershell
python -m http.server 4173
```

ブラウザで `http://localhost:4173` を開きます。スマホで公開する場合はHTTPSでホスティングしてください。

## ファイル構成

- `index.html` — 画面
- `styles.css` — レスポンシブデザイン
- `app.js` — 撮影、画像処理、トリミング、保存
- `manifest.webmanifest` / `sw.js` — PWA・オフライン対応
