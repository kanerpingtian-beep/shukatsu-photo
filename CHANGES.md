# Shukatsu Photo v5 改善内容

リポジトリの4ファイル（index.html / app.js / styles.css / sw.js）をそのまま上書きコミットしてください。キャッシュバスターは v=5 に更新済みです。

## 1. 背景切り抜きの品質改善（最重要）

白背景で髪・首・肩に出ていた違和感の主因は、半透明の輪郭ピクセルに元背景の色が混ざったまま合成されていたこと（色フリンジ）。以下を実装:

- **マスク収縮（erode）**: マスクを内側に約2px縮め、混色した境界帯を捨てる
- **フェザリング**: 収縮後の輪郭を分離ボックスブラーでなだらかに戻す
- **しきい値調整**: 人物領域の判定を 0.42 → 0.34 に下げ、髪の毛先の取りこぼしを軽減（アルファは smoothStep(0.34, 0.62)）
- 合成時の追いぼかしはマスク側で処理済みのため弱めた（首周りのゴースト軽減）

処理は写真1枚につき1回、約1秒（「AI認識中…」表示中に完了）。

## 2. 肌スムージングの人物限定化

従来は画像全体に blur がかかり、目・髪・スーツまでぼけていた。ぼかしレイヤーを人物マスクで切り抜いてから重ねる方式に変更。背景と輪郭のシャープさを保ったまま肌だけなめらかになる。

## 3. 新機能

- **セルフタイマー**: 撮影モードに なし/3秒/10秒。カウントダウン表示、再タップでキャンセル。撮影前チェックの文言とも整合
- **コンビニ印刷プリセット**: L判（89×127mm・300dpi = 1500×1051px）に 30×40mm を8枚面付け、切り取り線付き。`shukatsu-photo-print.jpg` で保存
- **JPEG に印刷解像度を埋め込み**: 履歴書プリセットは 762dpi を JFIF に書き込み → 印刷ソフトでちょうど 30×40mm になる（従来は「300dpi」表記が不正確＆メタデータなし）
- **ファイルサイズ表示**: 保存欄に「約1.2MB」等を表示（操作が落ち着いてから計測、900msデバウンス）
- **長押し比較ボタン**: 押している間だけ補正前の元写真を表示
- **縮小ズーム**: 拡大率の下限を 1.0 → 0.6 に。顔が大きく写った自撮りも枠に収められる

## 4. UI / その他

- 小さすぎたフォントを底上げ（9–10px → 11–12px）、スライダーのつまみを 14 → 22px、当たり判定を拡大、トグル・スウォッチも拡大
- AIモデル初回読込時は「AIモデル読込中…（初回のみ）」と表示
- **Service Worker**: AIモデル・WASM・フォントをキャッシュ優先に → 2回目以降はオフラインでも背景分離が動く
